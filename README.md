# Simple Interest Calculator

## Project Description
This project calculates simple interest based on user input.

## Author
Thatavarthi Shravya

## Formula
Simple Interest = (Principal × Rate × Time) / 100

```
Input:
   p, principal amount
   t, time period in years
   r, annual rate of interest
Output
   simple interest = p*t*r
```
